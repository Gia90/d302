# RFID D302

Python scripts to interact with the RFID Reader/Writer D302

## Install

```
virtualenv3 .env
source .env/bin/activate
pip install -r requirements.txt
```

## Usage

```
(sudo) python ./pollread.py -d /dev/ttyUSBXXX
```
